# 六.查找

## 一.ASL

关键字的平均比较次数，称为**平均查找长度**(ASL)。

**ASL=Σ^n^~i-1~ p~i~ c~i~**

**n**：记录的个数

**p~i~**：查找第i个记录的概率(一般认为是p~i~=1/n)

**c~i~**：找到第i个记录所需的比较次数

## 二.线性表的查找

顺序表或线性链表表示**静态查找表**，表内的元素无序。

### I.顺序查找

```C++
int SqListFind(SqList L,Keytype key){
    L.r[0]=key;
    for(i=L.length-1;L.r[i]!=key;i--);
    return i;
}
```

时间复杂度**T(n)=o(n)**
